# 🚜 FieldLog

**FieldLog** is a mobile-first field work logging application designed for **tractor and JCB operators** to track working hours, monitor field activity, and analyze productivity.

The app helps farmers, contractors, and machine operators efficiently **record machine usage, manage field work sessions, and review historical work data** in a clean and simple mobile interface.

---

# ✨ Features

## ⏱ Work Time Tracking
- Start and stop work sessions
- Automatically calculate working hours
- Track total machine usage time

## 📁 Field Work Records
- Log work done on specific fields
- Record machine activity
- Maintain organized work logs

## 📊 Dashboard Analytics
- View daily work statistics
- Track total hours worked
- Quick overview of productivity
- Save & Export PDF easily with simple one click by selecting which entries you want

## ▶ Work Session Logging
- Start a work session when work begins
- Stop the session when work ends
- Save logs automatically

## 📱 Mobile Friendly UI
- Clean modern mobile interface
- Smooth navigation
- Floating tab bar design

## 📜 History Management
- View past work sessions
- Track previous machine activity
- Access logs anytime

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/yshbtw/FieldLog.git
```

Move into the project folder

```bash
cd FieldLog
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npx expo start
```

---

# 📱 Running the App

### Run on Android
Press **A** in the terminal.

### Run on iOS
Press **I** in the terminal.

### Run using Expo Go
Scan the QR code displayed in the Expo DevTools.
